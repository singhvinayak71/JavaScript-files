Ajax

AJAX = Asynchronous JavaScript And XML.


AJAX just uses a combination of:

  • A browser built-in XMLHttpRequest object (to request data from a web server)
  • JavaScript and HTML DOM (to display or use the data)

step to use AJAX

  • create a html page and include JavaScript and css file
  • In HTML, create a button and give some id
    • in JavaScript file, add a eventListener like: fetchData.addEventListener('click', fetchxhr);
    • create an XMLHttpRequest object like: let data = new XMLHttpRequest();
    • then define function for click event like: function fetchxhr(){ }
    • In fetchxhr function, we can define a callback function like: data.onload = funcion(){ // add some code}
    • then send a request to server. you can use open() and then send() like: data.open('GET',"ajax2.txt"); data.send();
